== Description ==

ProGo Demo Bar plugin adds another bar, like the built-in (to WP v3.1+) Admin Bar, to the front-end of ProGo Themes sites, allowing visitors to explore different appearance options on the ProGo Theme.

Currently supports listing: Pages, Menus, and (Post) Categories

== Installation ==

1. Upload the entire `progo-demo-bar` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go to your homepage, or any other front-end page, to see the ProGo Demo Bar
