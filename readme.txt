=== ProGo Demo Bar ===
Contributors: chousmith
Requires at least: 3.1
Tested up to: 3.2.1
Stable tag: 0.5

Adds bar that sticks to top of front end of site, allowing visitors to preview different Color Schemes and other Layout options.

== Description ==

ProGo Demo Bar plugin adds another bar, like the built-in (to WP v3.1+) Admin Bar, to the front-end of ProGo Themes sites, allowing visitors to explore different appearance options in that ProGo Theme.

Currently supports listing: Pages, Menus, and (Post) Categories

== Installation ==

1. Upload the entire `progo-demo-bar` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go to your homepage, or any other front-end page, to see the ProGo Demo Bar
